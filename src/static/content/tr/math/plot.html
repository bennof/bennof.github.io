TabularRasa

Plotting

math/plot

ChartJS

TabularRasa uses ChartJSfor plotting. all ChartJS features an the object is available. The Chart object is returned by draw function.

Create a plot

To create a simple plot:


// Javascript
var plot = new TR.math.plot.Plot("Fallzahlen");
plot.x(TR.math.seq(-2,16)); // set x axis from -2 to 15
plot.line(TR.math.seq(-4,6,1),"Line Plot","#af5431"); // a red line
plot.fline(TR.math.seq(0,16,0.4),"Filled Line Plot","#3e95cd"); // blue line filled below
plot.x_marker(3,"X-Marker","#f1cf0b"); // vertical line at index 3
plot.y_marker(3,"Y-Marker","#8b61ad"); // horizontal line at value 3
plot.draw("tr_plot_example"); //draw to html canvas with id="tr_plot_example"

Output:

A plot can be redrawn by first calling plot.clear() an then calling draw method again.
