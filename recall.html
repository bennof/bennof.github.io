About

ReCall is a simple functional JS lib

The main goal ist to keep it simple and small.

  • recall.js

Ready

Execute javascript after document is loaded.

Here is a simple example:

 ReCall.Ready(function() {
    ReCall.FixedToolbar('#menubar');
    ReCall.CleanCode('.clean-code');
});

Fixed Toolbar

Auto Complete

AutoComplete for search fields etc.

Try to keep ist as simple as possible. Search for some countries.

Html:

Javascript:

ReCall.AutoComplete('#search_example',{
    hint: 'data/autocomplete.jsonp?hint=', 
    url: 'recall.html?s='
});

AJAX / JSONP

Unified Interface

AJAX and JSONP can be replaced by each other.

//JSONP
ReCall.JSONP.Request(url,{
    onSuccess: function(msg){ ... }, 
    timeout: 5
});

//AJAX
ReCall.AJAX.Request(url,{
    onSuccess: function(msg){ ... }, 
    timeout: 5
});
